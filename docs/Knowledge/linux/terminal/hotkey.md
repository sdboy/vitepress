# 终端快捷键

## 光标移动

| 快捷键 | 功能 |
| :--: | :-- |
| `Ctrl + A` | 移动到行首 |
| `Ctrl + E` | 移动到行尾 |
| `Ctrl + ←` / `→` | 按单词跳转 |
| `Ctrl + B` / `F` | 左右移动一个字符 |

## 编辑操作

| 快捷键 | 功能 |
| :--: | :-- |
| `Ctrl + U` | 删除光标前所有内容 |
| `Ctrl + K` | 删除光标后所有内容 |
| `Ctrl + W` | 删除光标前一个单词 |
| `Ctrl + Y` | 粘贴之前删除的内容 |
| `Ctrl + C` | 取消当前命令 |
| `Ctrl + D` | 退出当前shell/EOF |

## 历史命令

| 快捷键 | 功能 |
| :--: | :-- |
| `↑` / `↓` | 上/下翻阅历史命令 |
| `Ctrl + R` | 搜索历史命令 |
| `Ctrl + P` | 上一条命令 |
| `Ctrl + N` | 下一条命令 |
| `!!` | 执行上一条命令 |
| `!n` | 执行第n条命令 |

## 进程控制

| 快捷键 | 功能 |
| :--: | :-- |
| `Ctrl + Z` | 挂起当前进程 |
| `Ctrl + L` | 清屏 |
| `Ctrl + S` | 暂停输出 |
| `Ctrl + Q` | 恢复输出 |
